
HKEX Official Price Fetch Package
=================================

Files:
  fetch_hkex.py         - Script to fetch daily OHLC CSV from HKEX blob storage and save to latest_price.json
  config_template.json  - Example snippet to include in your main model config
  latest_price.json     - Output file generated by script (ignored initially)

Steps (macOS / Linux):
  1.  pip install requests
  2.  python fetch_hkex.py --code 02618 --date 2025-05-06
      (omit --date to use today's HK date)
  3.  Ensure your main prediction script loads OHLC from latest_price.json
      when use_official_ohlc = true.

Safety:
  - No API key needed for HKEX public CSV.
  - If HTTP 404, data for that date may not be published yet (fetch after 18:00 HKT).
