
HKEX Official OHLC Fetch (Standalone)
=====================================

Files
-----
fetch_hkex.py         - Download HKEX daily CSV and stores open/high/low/close in latest_price.json
latest_price.json     - Output file (generated by script)

Steps
-----
1. pip install requests
2. python fetch_hkex.py --code 02618            # today's HKT date
   python fetch_hkex.py --code 02618 --date 2025-05-06
3. The generated 'latest_price.json' can be read by your prediction script.

Notes
-----
• No API key needed: HKEX provides anonymous CSV endpoints.
• Run this script before executing your Monte Carlo prediction so that S₀ and σ
  can use fresh official data.
